# A simple Pet Daycare application for keeping track of your pets
 
 ## Features :
 - Authentication usign Next-Auth
 - Stripe payment
 - Access based on payment
 - Add multiple pets
 - Edit pets
 - Delete pets
 - Postgresql for database
 - Prisma for db interactions
 - Jwt for manage sessions
